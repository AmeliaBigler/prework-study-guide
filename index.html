HTML

  • HTML is a programming language.
  • HTML stands for HyperText Markup Language.
  • The head element contains information about the webpage. This content is not visible to the user.
  • The body element represents the visible content shown to the user.
  • Tags are the code inside angled brackets. Many (but not all) elements have opening and closing tags.

CSS

  • CSS is a programming language.
  • CSS stands for Cascading Style Sheets.
  • Types of CSS: in-line, internal, external.
  • If using external CSS, the link must be added to the HTML file metadata / head element.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a program used for version control of code.
  • Git Flow is the process of pulling and pushing code between the main branch and feature branches.
  • Git commands are entered in to the CLI.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a programming language.
  • A variable is a named container that allows us to store data in our code as values.
  • Types of values include string values, number values, and boolean values.
  • Control Flow is the order in which a computer executes code in a script.
